Course Syllabus


Introduction to Tableau:

1. Understand the role of Tableau in data analysis and visualization.
2. Install Tableau Desktop and set up your workspace.
3. Explore the Tableau interface and basic functionalities.

Connecting to Data Sources:

1. Connect to various data sources (Excel, databases, web data connectors).
2. Clean and transform data using Tableau Prep Builder.

Creating Visualizations:

1. Build essential charts (bar charts, line charts, scatter plots, etc.).
2. Customize visualizations with labels, colors, and formatting.
3. Use calculated fields and parameters for dynamic analysis.
Data Exploration and Analysis:

1. Filter data using context filters and quick filters.
2. Create sets and groups for comparative analysis.
3. Understand level of detail (LOD) expressions.

Dashboard Design:

1. Design interactive dashboards with multiple visualizations.
2. Add actions (filter actions, highlight actions) for user interaction.
3. Incorporate maps and geographic data.

Publishing and Sharing:

1. Publish workbooks to Tableau Server or Tableau Online.
2. Share dashboards with colleagues and stakeholders.
3. Schedule data refreshes.

Learning Objectives:

1. Navigate the Tableau interface confidently.
2. Create meaningful visualizations from raw data.
3. Analyze data subsets using filters and sets.
4. Design interactive dashboards for data exploration.
5. Publish and share insights with others.
