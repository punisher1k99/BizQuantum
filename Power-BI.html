Course Syllabus


Introduction to Power BI:

1. Understand the Power BI ecosystem and its components.
2. Install Power BI Desktop and connect to data sources.
3. Explore the query editor and data transformation techniques.

Basic Report Design: 

1. Create compelling visualizations using Power BI Desktop.
2. Master canvas layout, visual types, and formatting options.
3. Learn about tooltips, interactions, and export capabilities.

Visual Synchronization and Grouping: 

1. Work with slicers for dynamic filtering.
2. Understand visual sync limitations.
3. Group data for better insights.

Hierarchies and Filters: 

1. Create hierarchies for drill-down analysis.
2. Apply filters to focus on specific data subsets.

Modeling and DAX Functions (Level 1 and Level 2): 

1. Build relationships between tables.
2. Write DAX expressions for calculated columns and measures.
3. Explore advanced DAX functions.

Power BI Cloud and Service: 

1. Publish reports to Power BI Service.
2. Collaborate, share dashboards, and schedule data refresh.

Excel Integration and Row-Level Security (RLS): 

1. Integrate Power BI with Excel.
2. Implement RLS for secure data access.

Report Server and RDL: 

1. Deploy reports to Power BI Report Server.
2. Understand RDL (Report Definition Language).
